# 기본문법 - 태그
* 태그: 태그 안의 내용물을 설명해줌
* 모든 태그를 알고있을 필요는 없음


# 줄바꿈
* <br>: line break
* <p>: paragraph


# 속성(Attribute)
* 이미지 태그: <img src="test.jpg", width="100%">
* 여기서 src="test.jpg"과 같은 부분을 속성이라고 함


# 부모 자식과 목록
* <li>: list
* <ol>: ordered list
* <ul>: unordered list
* ol/ul 사이에 li태그 사용


# 문서의 구조와 슈퍼스타들
* <title>: 문서의 제목
* <meta charset="utf-8">: 문서가 utf-8로 인코딩 되어있음을 알려줌
* <head>: 본문을 설명하는 태그
* <body>: 본문
* <html>: 문서가 html로 작성되었음을 알려줌


# HTML 태그의 제왕
* <a>: anchor, 링크
* href: hypertext reference, 주소
